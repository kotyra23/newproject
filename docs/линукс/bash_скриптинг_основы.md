# 💻 Bash Скриптинг: Основы

## 🚀 Начало работы

### Создание первого скрипта

```bash
#!/bin/bash  # Shebang — указывает интерпретатор
echo "Hello World"

#Запуск:

# 1. Сделать исполняемым:
chmod +x script.sh

# 2. Запустить:
./script.sh

# Или без +x:
bash script.sh

## 📦 Переменные

#Объявление и использование


name="John"           # Присваивание (БЕЗ пробелов вокруг =)

echo $name            # Вывод значения

echo ${name}          # Безопасный вариант (рекомендуется)


Правила именования:

Только буквы, цифры, подчёркивание

Не может начинаться с цифры

Чувствительны к регистру: Name ≠ name


my_var="value"        # ✅ Правильно

my-var="value"        # ❌ Ошибка (дефис)

1var="value"          # ❌ Ошибка (начинается с цифры)


#Спецпеременные

| Переменная | Описание | Пример |
| :--- | :--- | :--- |
| `$0` | Имя скрипта | `./script.sh` |
| `$1`, `$2` | Позиционные аргументы | `./script.sh arg1 arg2` |
| `$#` | Количество аргументов | `3` (для трёх аргументов) |
| `$?` | Код возврата последней команды | `0` (успех) |
| `$@` | Все аргументы как список | `"$@"` (в цикле) |
| `$*` | Все аргументы как одна строка | `"${*}"` |
| `$$` | PID текущего процесса | `12345` |
| `$USER` | Имя текущего пользователя | `nik` |
| `$HOME` | Домашняя директория | `/home/nik` |
| `$PWD` | Текущая директория | `/home/nik/project` |

#Пример использования аргументов:


#!/bin/bash
echo "Скрипт: $0"

echo "Первый аргумент: $1"

echo "Всего аргументов: $#"

echo "Все аргументы: $@"

./script.sh hello world

# Вывод:

 Скрипт: ./script.sh

 Первый аргумент: hello

 Всего аргументов: 2

 Все аргументы: hello world


## 📥 Ввод данных (read)

Базовый ввод


read -p "Введите имя: " username

echo "Привет, $username!"

#Опции read

| Опция | Описание | Пример |
| :--- | :--- | :--- |
| `-p` | Подсказка (prompt) | `read -p "Name: " var` |
| `-s` | Скрытый ввод (пароль) | `read -s -p "Pass: " pass` |
| `-t N` | Таймаут N секунд | `read -t 10 var` |
| `-n N` | Читать только N символов | `read -n 1 key` |
| `-a` | Читать в массив | `read -a arr` |

#Пример ввода пароля:

read -s -p "Пароль: " password

echo ""  # Новая строка после ввода

echo "Пароль принят (длина: ${#password})"


##🔀 Условия (if/else)
Базовый синтаксис


if [ условие ]; then
    # код если истина
elif [ другое_условие ]; then
    # код если другое условие
else
    # код если всё ложно
fi

#Сравнение чисел

| Оператор | Описание | Пример |
| :--- | :--- | :--- |
| `-eq` | Равно (equal) | `[ $a -eq $b ]` |
| `-ne` | Не равно (not equal) | `[ $a -ne $b ]` |
| `-gt` | Больше (greater than) | `[ $a -gt $b ]` |
| `-lt` | Меньше (less than) | `[ $a -lt $b ]` |
| `-ge` | Больше или равно | `[ $a -ge $b ]` |
| `-le` | Меньше или равно | `[ $a -le $b ]` |

#!/bin/bash
age=20

if [ $age -ge 18 ]; then
    echo "Взрослый"
elif [ $age -lt 18 ]; then
    echo "Ребенок"
else
    echo "Ошибка"
fi

#Сравнение строк

| Оператор | Описание | Пример |
| :--- | :--- | :--- |
| `=` | Равно | `[ "$a" = "$b" ]` |
| `!=` | Не равно | `[ "$a" != "$b" ]` |
| `-z` | Пустая строка | `[ -z "$var" ]` |
| `-n` | Не пустая строка | `[ -n "$var" ]` |

#!/bin/bash
name="Alice"

if [ -z "$name" ]; then
    echo "Имя не введено!"
elif [ "$name" = "Admin" ]; then
    echo "Привет, админ!"
else
    echo "Привет, $name!"
fi


#Проверка файлов и директорий

| Оператор | Описание | Пример |
| :--- | :--- | :--- |
| `-f` | Файл существует | `[ -f file.txt ]` |
| `-d` | Директория существует | `[ -d /tmp ]` |
| `-e` | Существует (файл или папка) | `[ -e path ]` |
| `-r` | Доступен для чтения | `[ -r file.txt ]` |
| `-w` | Доступен для записи | `[ -w file.txt ]` |
| `-x` | Исполняемый | `[ -x script.sh ]` |


Пример:

#!/bin/bash
file="config.txt"

if [ -f "$file" ] && [ -r "$file" ]; then
    echo "Файл $file существует и доступен для чтения"
    cat "$file"
else
    echo "Ошибка: файл $file не найден или недоступен"
    exit 1
fi

#🔢 Арифметика
Способы вычислений

# 1. Двойные скобки (рекомендуется):
result=$(( 5 + 3 * 2 ))    # 11
(( result++ ))             # Инкремент

# 2. let:
let x=10
let x=x+5                  # 15

# 3. expr (устаревший):
result=$(expr 10 + 5)      # 15

##🧩 Логические операторы
В условиях

| Оператор | Описание | Пример |
| :--- | :--- | :--- |
| `&&` | И (AND) | `[ -f file ] && cat file` |
| `\|\|` | ИЛИ (OR) | `[ -f file ] \|\| echo "Нет файла"` |
| `!` | НЕ (NOT) | `[ ! -f file ]` |


#Примеры:

# Выполнить команду только если файл существует:
[ -f config.txt ] && echo "Файл найден"

# Или вывести ошибку если файла нет:
[ -f config.txt ] || echo "Ошибка: файл не найден!" >&2

# Комбинирование:
[ -f file.txt ] && [ -r file.txt ] && cat file.txt

# Отрицание:
[ ! -d backup ] && mkdir backup

Пример: Проверка аргументов

#!/bin/bash

# Проверка количества аргументов
if [ $# -ne 2 ]; then
    echo "Использование: $0 <имя> <возраст>"
    exit 1
fi

name=$1
age=$2

# Проверка что возраст — число
if ! [[ $age =~ ^[0-9]+$ ]]; then
    echo "Ошибка: возраст должен быть числом!"
    exit 1
fi

echo "Привет, $name! Тебе $age лет."

Пример 2: Резервное копирование

#!/bin/bash
source_dir="/home/user/docs"
backup_dir="/backup/docs_$(date +%Y%m%d)"

if [ ! -d "$source_dir" ]; then
    echo "Ошибка: исходная папка не найдена!"
    exit 1
fi

mkdir -p "$backup_dir"
cp -r "$source_dir"/* "$backup_dir/"

if [ $? -eq 0 ]; then
    echo "✅ Бэкап создан: $backup_dir"
else
    echo "❌ Ошибка при копировании!"
    exit 1
fi
