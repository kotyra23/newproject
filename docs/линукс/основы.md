## 🧭 Навигация

### Команда `pwd`

**Описание:** Показывает текущую папку (Print Working Directory).

```bash
pwd
# /home/user

Команда cd
Описание: Переход между папками (Change Directory).


cd /etc              # Абсолютный путь

cd ..                # На уровень вверх

cd ~                 # В домашнюю папку

cd /var/log          # Полный путь к папке


Специальные пути:

. — текущая папка

.. — родительская папка

~ — домашняя директория

Команда ls

Описание: Содержимое директории (List).


ls           # Простой список

ls -l        # Детальный вывод

ls -a        # Показать скрытые файлы

ls -la       # Детально + скрытые

Флаги:

-l — подробный список

-a — скрытые файлы

-t — по времени

-S — по размеру

-r — в обратном порядке

## 📂 Детали `ls -l`

**Пример вывода:**


drwxr-xr-- 2 root root 4096 Jan 10 12:00 apache2


## Разбор прав доступа:

1. **Первый символ** — тип:
   
   - `d` — папка (directory)
   
   - `-` — файл
   
   - `l` — ссылка

2. **Следующие 9 символов** разбиты на 3 группы:
   
   - **Владелец** (user): `rwx`
   
   
   - **Группа** (group): `r-x`
   
   - **Остальные** (others): `r--`

### Права:
- **r** (read) — чтение
- **w** (write) — запись
- **x** (execute) — исполнение

**Пример:**
- `rwx` — владелец может читать, писать, исполнять
- `r-x` — группа может читать и исполнять
- `r--` — остальные только читают

## 🗑️ Удаление файлов

### Команда `rm`

**Описание:** Удаление (Remove).

```bash
rm file.txt           # Удалить файл

rm -r folder          # Удалить папку рекурсивно

rm -rf folder         # Принудительно без вопросов

---


Флаги:

-r — удаление папок (рекурсивно)

-f — удаление без подтверждения (force)

## 🔍 Поиск и просмотр

Команда grep

Описание: Поиск текста по шаблону.

grep "pattern" file.txt

Команда find

Описание: Поиск файлов по имени/типу.

find . -name "*.txt"

Команда less / more

Описание: Постраничный просмотр.


less bigfile.log
