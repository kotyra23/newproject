## 📝 Текстовые редакторы

### Nano
Простой редактор.
- `Ctrl+O` — записать файл.
- `Ctrl+X` — выход.
- `Ctrl+W` — поиск.

### Vim / Vi
Мощный модальный редактор.

**Режимы:**
- **Normal** (командный): `ESC`. Навигация `h/j/k/l`.
- **Insert** (вставка): `i`. Ввод текста.
- **Visual** (выделение): `v`.

**Основные команды:**
- `:w` — сохранить.
- `:q!` — выйти без сохранения.
- `dd` — удалить строку.
- `x` — удалить символ.
- `:s/old/new/g` — замена текста.

## 🔒 Права доступа

### chmod (изменение прав)
Символьный режим: `u` (user), `g` (group), `o` (others), `a` (all).
Операторы: `+` (добавить), `-` (убрать), `=` (установить).
Права: `r` (read), `w` (write), `x` (execute).

**Примеры:**
```bash
chmod u+x script.sh   # Добавить исполнение владельцу
chmod 755 file        # rwxr-xr-x (числовой режим)
chmod -R 777 folder   # Рекурсивно всем всё (небезопасно!)
